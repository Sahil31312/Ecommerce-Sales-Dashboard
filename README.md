# Power BI Sales Dashboard Project

## Project Overview
Built an interactive **Power BI dashboard** to track and analyze online sales data, providing clear insights into business performance.

## Key Features

### Data Modeling
- Created relationships between multiple data sources
- Used joins to integrate and clean datasets
- Developed calculated columns and measures for advanced analysis

### Data Exploration
- Implemented filters and slicers for dynamic data exploration
- Enabled drill-through functionality for detailed insights
- Supported user-driven parameters for interactive analysis

### Visualizations
Designed a variety of Power BI visuals, including:
- Bar charts
- Line charts
- Pie charts
- Donut charts
- Clustered bar charts
- Scatter plots
- Area charts
- Map visualizations
- Interactive slicers

## Outcome
Delivered a dynamic and interactive dashboard that helps analyze sales trends, monitor performance, and support data-driven decision-making.
